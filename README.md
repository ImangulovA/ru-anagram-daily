# Анаграмма дня (ru-anagram-daily)

Ежедневная игра в слова на русском. Две подсказки дают два исходных слова
(**А** и **Б**); из всех их букв складывается третье слово (**В**, ответ). Слова
А и Б вводятся в кроссвордные клетки, буквы попадают в перемешиваемый **пул
букв**, из которого собирается финальный ответ. Подсказки стоят очков.

Русская версия английской [Anagram Daily](https://github.com/ImangulovA/anagram-daily),
форк платформы [`daily_github_game`](https://github.com/ImangulovA/daily_github_game):
общие шапка/подвал, тема, архив, статистика и глобальная статистика с другими
ежедневными играми (Train Tracks, Battleships).

Live: https://imangulova.github.io/ru-anagram-daily/ · Все игры: https://imangulova.github.io/games/

## Как играть

- Разгадайте **Подсказку 1** и **Подсказку 2** в слова А и Б. **Проверить** на
  карточке тестирует введённое (стоит очков).
- Застряли? **Открыть букву** или **Открыть слово** на карточке (стоит очков).
- **Пул букв** показывает все буквы А + Б. **Перемешать** бесплатно. Нажимайте на
  плитки, чтобы собрать слово В.
- **Ответить** проверяет финал. Неверные попытки трясут клетки и вычитают очки.
  **Открыть букву** / **Открыть подсказку** помогают с В; **Сдаться** обнуляет день.
- Каждый день начинается с **20.0** очков (не ниже 0). На финальном экране: счёт,
  три слова с подсказками, глобальная статистика с гистограммой, кнопка
  **Поделиться** и обратный отсчёт.
- **Архив** / **Статистика** в шапке -- переиграть прошлые дни и увидеть итоги.

## Источники данных

- **Слова + частота**: пакет `wordfreq` (русские корпуса, офлайн) -- пул
  слов-кандидатов и частота для сложности/курации.
- **Определения (подсказки)**: помощник кроссвордиста «Словарь кроссвордиста»
  graycell.ru (`graycell.ru/word/<слово>`) -- корректные авторские кроссвордные
  определения. Слово попадает в словарь, только если у graycell есть хотя бы одна
  подсказка, что заодно отфильтровывает пул до реальных ответных слов.

## Структура репозитория

| Путь | Роль |
|---|---|
| `app/` | SvelteKit-приложение (форк). `src/lib/game/` -- код игры; `src/lib/platform/` -- общая платформа. |
| `app/src/lib/game/GameComponent.svelte` | UI головоломки (карточки подсказок, пул букв, финальный ответ). |
| `app/src/lib/game/scoring.js` | Чистый движок подсчёта (кириллица; хранение -- на платформе). |
| `app/src/lib/game/data/days.js` | АВТОГЕНЕРАЦИЯ данных (`dayIndexes()`/`loadDay()`). |
| `app/src/lib/game/index.js` | Конфиг `GAME` (id, anchor, scoreOf, shareLine). |
| `scripts/build_words.py` | Сбор слов + определений (wordfreq + graycell) -> `data/words_defs.json`. |
| `scripts/gen_puzzles.py` | Движок анаграмм + генератор дней. |
| `scripts/verify_puzzles.py` | Обязательный гейт инвариантов. |
| `data/` | Входы/выходы генератора (`words_defs.json`, `puzzles.json`). |

## Идентификаторы (важно)

- **Ключ хранения**: `ruanagram_day<N>` (`GAME.id = 'ruanagram'`) -- отдельно от
  английской игры, читается хабом как своя запись.
- **Ключ статистики**: `ru-anagram-daily` (`GAME.statsId`) -- отдельный воркер/таблица.
- **День 0** = локальная полночь **2026-07-06** (`GAME.anchorDate = [2026, 6, 6]`),
  день считается через `Math.round` (устойчиво к переводу часов), как в хабе.

## Правила слов

- Алфавит кириллический; **Ё -- отдельная строгая буква** (без свёртки ё->е) в
  подписи анаграммы.
- Длина ответа В по дням недели: Пн/Вт/Вс -- 9, Ср/Чт -- 10, Пт/Сб -- 11..13.
- Инвариант: `sorted(А + Б) == sorted(В)`; слова А/Б/В не однокоренные; исходное
  слово не является подстрокой ответа.

## Разработка / сборка

Node не системный: `export PATH="$HOME/.local/node/bin:$PATH"`.

```bash
cd app
npm install          # в CI; локально node_modules переиспользуется из форка
npm run dev          # локальный сервер
BASE_PATH=/ru-anagram-daily npm run build   # прод-сборка -> app/build
```

## Пересборка данных и головоломок

Скрипты запускаются интерпретатором из соседнего форка (там есть wordfreq/nltk):

```bash
PY=~/Desktop/anagram-daily/.venv/bin/python
$PY scripts/build_words.py --limit 25000     # -> data/words_defs.json (кеш: data/graycell_cache.json)
$PY scripts/gen_puzzles.py                    # -> data/puzzles.* И app/src/lib/game/data/days.js
$PY scripts/verify_puzzles.py                 # обязательный гейт
```

## Деплой

GitHub Actions (`.github/workflows/deploy.yml`) собирает `app/` и публикует
`app/build` в GitHub Pages при пуше в `main`. Источник Pages -- **GitHub Actions**.
`BASE_PATH` выводится из имени репозитория (`/ru-anagram-daily`).

## Стиль

Необрутализм (общий с играми-сиблингами): тёплый светлый по умолчанию + тёмная
тема, акцент `#fdc800`, жирные рамки, жёсткие тени, Inter + JetBrains Mono.
